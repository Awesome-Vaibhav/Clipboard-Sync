# 📋 Clipboard Sync

A lightweight Android clipboard manager that automatically saves and organizes copied text for quick access.

---

## ✨ Features

- 🔄 Automatic clipboard monitoring
- 📌 Pin important snippets
- 🗑️ Swipe to delete entries
- ⚡ One-tap copy & share
- 🔍 Real-time search
- 💾 Offline-first storage using Room Database
- 🌙 Clean dark theme UI

---

## 📱 About

Clipboard Sync automatically tracks copied text and stores it locally on your device.

Whether you're saving links, notes, code snippets, or frequently used text, Clipboard Sync keeps everything organized and easily accessible.

---

## 🛠️ Built With

- Kotlin
- Jetpack Compose
- Material 3
- Room Database
- Coroutines
- MVVM Architecture

---

## 🔒 Privacy

Clipboard Sync works completely offline.

- No cloud storage
- No analytics
- No tracking
- No data collection

Everything stays on your device.
