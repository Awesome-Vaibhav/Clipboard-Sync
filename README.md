🌟 About The Project

Clipboard Sync is an elegant offline-first Android utility designed for users who frequently copy text snippets and need instant access to clipboard history.

Whether it's code snippets, URLs, passwords, notes, OTP references, or frequently used text, Clipboard Sync ensures everything remains organized and instantly accessible.

The application combines:

🎨 Modern Material 3 Design

⚡ Instant Clipboard Tracking

📌 Pin & Organize Features

🔍 Real-Time Search

🗂 Local Room Database Storage

🌙 Beautiful Cosmic Slate Dark Theme



---

🎯 Key Functional Capabilities

🔄 Reliable Background Syncing

Automatically monitors clipboard changes and instantly saves copied text snippets, including entries copied from Gboard and other input methods.

🗑️ Ergonomic Swipe-to-Delete

Delete any clipboard entry with smooth swipe-left or swipe-right gestures.

📌 Pinning & Organization

Pin important snippets to keep them permanently at the top of your history.

⚡ One-Tap Quick Actions

Instantly:

Copy old snippets back to clipboard

Share snippets using Android Share Sheet

Access frequently used entries


🔍 Real-Time Search & Filtering

Find any clipboard item instantly while typing.
